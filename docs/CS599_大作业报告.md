# 企业级软件设计与开发课程大作业报告

## 项目名称
UAV-Mission-Agent：面向多无人机任务分配的多智能体协作 Agent 系统

## 1. 摘要
本项目设计并实现了一个 Agentic AI 原生的多无人机任务分配系统。系统通过任务分析 Agent、资源调度 Agent、路径规划 Agent 和执行监控 Agent 的协作，完成从任务输入到调度方案输出的端到端流程。

## 2. 问题背景
多无人机任务分配常见于应急救援、物流配送、巡检监控等场景。传统方法往往依赖人工规则或单一优化算法，难以解释复杂任务约束下的多步骤决策。本项目引入多智能体协作框架，将复杂任务拆解为多个可解释的 Agent 决策阶段。

## 3. 系统设计
系统采用分层架构：入口编排层、Agent 协作层、工具层、记忆层、评估层。

## 4. 核心实现
- `TaskAnalyzerAgent`：任务解析、优先级判断、子任务分解。
- `FleetManagerAgent`：无人机注册表、能力匹配、资源优化。
- `RoutePlannerAgent`：路径生成、障碍物规避、禁飞区规避。
- `MonitorAgent`：执行模拟、异常检测、建议生成。
- `VectorMemory`：历史任务经验存储。

## 5. 评估方法
评估指标包括任务成功率、总代价、平均风险、资源利用率和协作效率。

## 6. Demo 展示流程
1. 运行 `python -m src.main`。
2. 输入或使用默认灾难救援任务。
3. 查看任务分析、无人机分配、路径规划和监控结果。
4. 查看生成的 JSON 调度方案和日志。

## 7. 总结与展望
当前版本是可运行 MVP。后续可引入 LangGraph 的显式状态图、真实 LLM Function Calling、MARL 调度策略、GNN 路径规划，以及真实地图和天气 API。
