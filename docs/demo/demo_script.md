# 5 分钟 Demo 脚本

## 0:00 - 0:40 项目介绍
本项目是一个面向多无人机任务分配的 Agentic AI 系统，目标是展示多智能体协作如何解决企业级复杂调度问题。

## 0:40 - 1:30 架构说明
展示 README 架构图，说明四个 Agent：任务分析、资源调度、路径规划、执行监控。

## 1:30 - 3:30 运行演示
执行：
```bash
python -m src.main
```
解释输出中的成功率、总代价、平均风险和保存的调度方案。

## 3:30 - 4:30 代码结构
快速打开 src/agents，说明每个 Agent 的职责；打开 docs/specs，说明 SDD 文档。

## 4:30 - 5:00 总结
强调系统覆盖课程要求：多智能体协作、状态管理、工具调用、记忆机制、可观测性和评估。
